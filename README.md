# Personal Site

## Overview
This repository contains the source code for my personal website. The site showcases my projects, résumé and blog posts.

## Features
- Responsive design built with HTML, CSS and JavaScript.
- Portfolio section with links to GitHub projects.
- Blog powered by Markdown.
- Contact form for enquiries.
