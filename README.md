## 🔍 Overview

This is my **personal digital presence** — a responsively designed portfolio and blog built from the ground up with modern web technologies. The site serves as a hub for:

- 📊 **Research & Projects** — Applied ML, econometrics, and development economics work
- 📝 **Blog & News** — Insights on migration, commodity markets, agricultural economics
- 📋 **CV & Experience** — Academic background, skills, and professional journey
- 💼 **Contact** — Reach out for collaborations, inquiries, or discussions

The site is **fully responsive**, accessible, and optimised for readability across all devices.

🚀 Deployment
The site is live at: ayodeleid.com

Hosted via GitHub Pages with a custom domain configured through CNAME file.

Deployment is automatic: Push to main branch → GitHub Pages builds & deploys instantly.

💻 Local Development
Prerequisites
Node.js (v16+) and npm
Setup & Run
bash
# Clone the repository
git clone https://github.com/AyodeleID/personal-site.git
cd personal-site

# Install dependencies
npm install

# Start local development server
npm start
The site will be available at http://localhost:8000 (or configured port).

Development Workflow
bash
# Make edits to HTML/CSS/JS files
# Live reload should automatically refresh your browser
# Commit changes to main branch to deploy
git add .
git commit -m "Update: [describe change]"
git push origin main
📖 Content Management
Adding a Blog Post
Create a new .html file in the root or posts/ directory
Follow the existing HTML structure (header, content section, footer)
Link it in news.html
Push to GitHub
Updating CV
Update Ayodele_Idowu_CV.pdf in the root
Or edit cv.html directly for online version
Modifying Projects
Edit projects.html to add/remove portfolio entries with GitHub links.
