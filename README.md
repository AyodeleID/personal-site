## 🔍 Overview

This repository contains the source code for my personal site: **[ayodeleid.com](https://ayodeleid.com)**.

The website is designed to be clean, professional, and content-focused, showcasing:

- My background and profile
- Projects and technical work
- Research interests and publications
- News/updates and professional activities
- Downloadable CV

## 💻 Local Development

Because this is a static website, you can run it locally in multiple ways.

### Option 1 — Open directly
- Open `index.html` in your browser.

### Option 2 — Use a local server (recommended)

```bash
# Python 3
python -m http.server 8000
# then open http://localhost:8000
```

### Option 3 — Node-based server (if preferred)

```bash
npx serve .
# or
npx http-server .
```
