## Local Development Setup
To run this project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/AyodeleID/personal-site.git
   cd personal-site
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
